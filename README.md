#/ NOTICE / ->             
--------------------------
Colors :                  
- C : Base orange         
- V : Base verte          
- B : Base bleue          
- N : Dark purple         
- M : Surprise            
- K : Change color inside 

Options Zoom :            
- Z - Souris 1 : Zoom In  
- X - Souris 2 : Zoom Out 
- Or use mouse wheel      

Mouvements :              
- Use Arrows              

Autres     :              
- R : Reset               
- H : Hide infos          
- +/- : +/- 10 iterations 
- 1>0 : Change ur fractal 
- SPACE : Bloquer Julia(2)
